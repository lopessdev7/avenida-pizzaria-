# 🍕 Avenida Pizzaria

Site institucional desenvolvido para apresentar a Avenida Pizzaria, valorizando a identidade da marca e proporcionando uma experiência moderna para os clientes.

---

## 🧠 Visão do Projeto

O projeto foi desenvolvido com foco em um design moderno, responsivo e intuitivo, facilitando a navegação e destacando os principais produtos e informações da pizzaria.

---

## ✨ Solução Entregue

- Desenvolvimento de site institucional responsivo
- Interface moderna e agradável
- Navegação simples e intuitiva
- Layout otimizado para celulares e computadores
- Organização das informações da empresa

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎯 Objetivo

Criar uma presença digital profissional para a pizzaria, transmitindo credibilidade, fortalecendo a marca e facilitando o contato com os clientes.
